# fable5-lp

Claude Fable 5 のランディングページです。

以下のようにClaude Fable 5が利用可能になったので、試してみました。

![alt text](image.png)


## 使用したプロンプト

```
以下のページを参照して、claude fable5の位置付けやこれまでのモデルと異なるところ、セールスポイントをLPにまとめてください。LPはテックで先進的なイメージにして、Javascriptなどを使用して動きのあるものにしてください。

https://www.anthropic.com/news/claude-fable-5-mythos-5
```

出力されたLPに対して、さらに以下を追加。

```
フリーの画像を使用したりして、もっとサイバーパンクな感じにしてください。
```
